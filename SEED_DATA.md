# Создание тестовых данных

## Запуск скрипта для создания тестовых данных

```bash
npm run seed
```

## Создаваемые тестовые данные

### Администратор
- **Логин:** `admin`
- **Пароль:** `admin123`
- **Роль:** Администратор

### Магазин одежды "Мода"
- **Тип:** Одежда (clothing)
- **Владелец:**
  - Логин: `owner_clothing`
  - Пароль: `owner123`
- **Продавец:**
  - Логин: `seller_clothing`
  - Пароль: `seller123`
- **Настройки:** Размеры включены, вес отключен

### Продуктовый магазин "Свежесть"
- **Тип:** Продукты (grocery)
- **Владелец:**
  - Логин: `owner_grocery`
  - Пароль: `owner123`
- **Продавец:**
  - Логин: `seller_grocery`
  - Пароль: `seller123`
- **Настройки:** Размеры отключены, вес включен

### Универсальный магазин
- **Тип:** Общий (general)
- **Продавец:**
  - Логин: `seller_general`
  - Пароль: `seller123`
- **Настройки:** Размеры и вес включены

## Использование

После запуска скрипта вы можете:

1. **Войти в приложение продавца:**
   - Используйте `seller_clothing`, `seller_grocery` или `seller_general`
   - Пароль: `seller123`

2. **Войти в приложение владельца:**
   - Используйте `owner_clothing` или `owner_grocery`
   - Пароль: `owner123`

3. **Войти в веб-панель администратора:**
   - Используйте `admin`
   - Пароль: `admin123`

## Важно

⚠️ Эти учетные данные предназначены только для тестирования!
В продакшене обязательно измените все пароли на безопасные.
