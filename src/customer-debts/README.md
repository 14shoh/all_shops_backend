# Customer Debts Module (Долги клиентов)

## Описание
Модуль для управления долгами клиентов. Позволяет продавцам и владельцам магазинов отслеживать, кто взял в долг, сколько, когда и за что.

## API Endpoints

### POST /customer-debts
Создать новый долг клиента
```json
{
  "customerName": "Имя клиента",
  "amount": 100.50,
  "description": "За что взял в долг",
  "debtDate": "2026-01-24",
  "shopId": 1
}
```

### GET /customer-debts
Получить все долги клиентов для текущего магазина
- Автоматически использует shopId из токена пользователя
- Кешируется на 60 секунд

### GET /customer-debts/total
Получить общую сумму всех долгов клиентов
- Кешируется на 60 секунд

### GET /customer-debts/:id
Получить конкретный долг по ID

### PATCH /customer-debts/:id
Обновить долг клиента

### DELETE /customer-debts/:id
Удалить долг клиента (soft delete)

## Кеширование
- Список долгов кешируется на 60 секунд
- Общая сумма кешируется на 60 секунд
- Кеш автоматически инвалидируется при создании/обновлении/удалении
