# Supplier Debts Module (Долги фирмам)

## Описание
Модуль для управления долгами перед поставщиками (фирмами). Позволяет владельцам магазинов отслеживать общую сумму долгов, сколько отдано и сколько осталось.

## API Endpoints

### POST /supplier-debts
Создать новый долг перед фирмой
```json
{
  "supplierName": "Название фирмы",
  "totalDebt": 5000.00,
  "paidAmount": 1000.00, // опционально
  "shopId": 1
}
```

### GET /supplier-debts
Получить все долги перед фирмами для текущего магазина
- Автоматически использует shopId из токена пользователя
- Кешируется на 60 секунд

### GET /supplier-debts/summary
Получить сводку по долгам:
```json
{
  "totalDebt": 10000.00,
  "totalPaid": 3000.00,
  "totalRemaining": 7000.00
}
```
- Кешируется на 60 секунд

### GET /supplier-debts/:id
Получить конкретный долг по ID

### PATCH /supplier-debts/:id
Обновить долг перед фирмой

### POST /supplier-debts/:id/payment
Добавить платеж по долгу
```json
{
  "amount": 500.00
}
```

### DELETE /supplier-debts/:id
Удалить долг перед фирмой (soft delete)

## Кеширование
- Список долгов кешируется на 60 секунд
- Сводка кешируется на 60 секунд
- Кеш автоматически инвалидируется при создании/обновлении/удалении/добавлении платежа
